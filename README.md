## Features

- kubernetes 환경 구성을 위한 vagrant configuration files
    - case using `kubeadam`
    - case using `kubespray`
- 그 외 zsh, [modern unix](https://github.com/ibraheemdev/modern-unix), docker install 등 linux 최초 설정을 위한 shell scripts 다수 포함


## Getting Started

```sh
cd ./vm/k8s-kubespray
vagrant up
```
